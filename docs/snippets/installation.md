Install globlin from [PyPI]:

[PyPI]: https://pypi.org/project/globlin/

=== "pip"
    ```console
    $ python3 -m venv .venv
    $ source .venv/bin/activate
    $ pip install globlin
    ```
=== "uv"
    ```console
    $ uv init
    $ uv add globlin
    ```
